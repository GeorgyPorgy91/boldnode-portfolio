# Boldnode Portfolio
A collection of AI automation workflows built with n8n.
Each workflow solves a real business problem — from lead qualification
to autonomous invoice processing.

## Workflows

| Workflow | Description | Tools |
|---|---|---|
| [Lead Qualifier](./lead-qualifier) | Autonomous lead qualification and routing | n8n, Gemini, Google Sheets, Gmail |
| [L&D Trend Watcher](./ld-trend-watcher) | Daily trend scanning and content generation | n8n, Gemini, Slack, Pollinations AI |
| [Intake Automation](./intake-automation) | AI-driven client intake from form to CRM | n8n, Gemini, Google Forms, Sheets, Slack |
| [WhatsApp Support Hub](./whatsapp-support-hub) | 24/7 dual-agent customer support with human-in-the-loop | n8n, WhatsApp Business API, Gemini, Telegram |
| [Voice-to-Action](./voice-to-action) | Voice note to calendar, Notion and Slack simultaneously | n8n, Telegram, OpenAI Whisper, GPT-4o-mini, Google Calendar, Notion, Slack |
| [Invoice Processing](./invoice-processing) | Autonomous accounts payable with human approval flow | n8n, GPT-4o Vision, Gmail, Airtable, Slack, Moneybird |
| [M&A Scraping Engine](./ma-scraping-engine) | Due diligence via multi-platform scraping and AI sentiment analysis | n8n, SerpAPI, Apify, GPT-4o, Airtable, Google Docs, Slack |

## Tech Stack

**Automation:** n8n, Webhooks, REST APIs
**AI:** OpenAI GPT-4o, Google Gemini, OpenAI Whisper
**Data:** Pinecone, Google Sheets, Airtable, Notion
**Frontend:** Lovable, Tally
**Integrations:** Gmail, Slack, WhatsApp Business API, Telegram, Google Calendar, Moneybird
